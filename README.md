## Usage

1. 创建`.env`文件，填写百度千帆大模型平台和阿里云百炼的密钥，参考以下格式：
```
QIANFAN_ACCESS_KEY=Your_Access_Key
QIANFAN_SECRET_KEY=Your_Secret_Key

ALI_API_KEY=sk-xxx
```

2. 安装依赖：
```bash
npm install
```

3. 运行应用：
```bash
npm run start
```
